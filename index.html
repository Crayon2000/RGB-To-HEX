RGB To HEX

Description

Ce programme vous permet de connaître une couleur en hexadécimal à partir du code RGB (Rouge, Vert, Bleu). À l'aide des "slider" il est aussi possible de trouver la couleur que l'on cherche. C'est très pratique quand on veut connaître le code d'une couleur pour mettre sur sa page Web. Une fois la couleur trouvée, on a qu'à appuyer sur le bouton "Copy to Clipboard" pour l'enregistrer et ensuite le coller à l'endroit approprié. Pour aller plus vite, on peut se servir des presets. En cliquant avec le bouton droit de la souris sur la zone de couleur, on peut ouvrir la fenêtre de couleur standard. Il est aussi possible de cliquer sur "Pick Color" pour prélever une couleur sur l'écran.
